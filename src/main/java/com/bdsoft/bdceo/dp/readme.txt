一，创建性模式
    Abstract Factory（抽象工厂）
    Builder（构建器）
    Factory Method(工厂方法）
    Prototype（原型）
    Singleton（单利）

二，结构性模式
    Adapter（适配器）
    Bridge（桥接）
    Composite（组合）
    Decorator（装饰）
    Facade（门面、外观）
    Flyweight（享元）
    Proxy（代理）

三，行为性模式
    Chain of Responsibility（责任链）
    Command（命令）
    Interpreter（解释器）
    Iterator（迭代器）
    Mediator（中介者）
    Memento（备忘录）
    Observer（观察者）
    State（状态）
    Strategy（策略）
    Template Method（模板方法）
    Visitor（访问者）
